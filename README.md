# 🎮 Tic-Tac-Toe

A simple and interactive **Tic-Tac-Toe** game built using **HTML, CSS, and JavaScript**. Challenge a friend in this classic game of X's and O's and enjoy a clean, responsive user interface.

## 📌 Features

* Interactive 3×3 game board
* Two-player gameplay
* Automatic win detection
* Draw detection
* Reset/New Game functionality
* Responsive design
* Clean and user-friendly interface

## 🛠️ Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)

## 🎯 How to Play

1. Player 1 starts with **X**.
2. Player 2 plays with **O**.
3. Players take turns selecting empty cells.
4. The first player to align three symbols horizontally, vertically, or diagonally wins.
5. If all cells are filled without a winner, the game ends in a draw.

## 🧠 Game Logic

* Tracks player turns.
* Validates moves.
* Checks all possible winning combinations.
* Displays winner or draw message.
* Allows restarting the game.
